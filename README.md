# Линейна филтрация на изображения

Програма за прилагане на линейна филтрация върху растерни изображения с възможност за промяна на коефициентите.

## Инсталация

Необходими библиотеки:
```bash
pip install pillow numpy
```

## Стартиране

```bash
python main.py
```

## Използване

1. Натиснете "Зареди изображение" и изберете файл (PNG, JPG, BMP, GIF)
2. Въведете коефициентите на филтърното ядро (5×5 таблица) или изберете предефиниран филтър
3. Задайте Scale и Offset стойности
4. Натиснете "Приложи филтър" за да видите резултата
5. Запазете резултата с бутона "Запази резултат"

## Формула

За всеки пиксел програмата прилага:
```
result = (Σ(pixel[i,j] × kernel[i,j])) / scale + offset
```

Всеки пиксел и неговите 24 съседа се умножават по съответните тегла от ядрото, сумират се, делят на scale и се добавя offset.

## Предефинирани филтри

- Identity - без промяна
- Blur - размиване
- Sharpen - изостряне
- Edge Detect - откриване на ръбове
- Emboss - релефен ефект
- Gaussian - Гаусово размиване
